# Malaria-Typhoid Expert System

A diagnostic expert system built with Flask to assist in identifying Malaria, Typhoid, and co-infections based on symptom analysis.

## Features
- Symptom checker
- Rule-based inference engine
- Admin interface for managing symptoms and rules
- User authentication

## Setup
1. `pip install -r requirements.txt`
2. `python manage.py db upgrade`
3. `python manage.py run`
